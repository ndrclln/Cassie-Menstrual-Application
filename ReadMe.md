# Cassie by Endrea
## A menstrual cycle tracker.
#### Final project of Mr. Arjonel Mendoza on our CC 102 - Advanced Computer Programming.


#### Description:
    - This is a application was created using tkinter and python language.
    - Created on March to April 2026.

#### App Features
    This application can:
    1. Add record for menstruation.
    2. Predict the next expected menstruation based on calculated cycle dates.
    3. Check for menstrual cycle regularity.
